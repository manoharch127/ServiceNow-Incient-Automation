# ServiceNow Incident Automation System

## 🚀 Overview
This project simulates a real-world ITSM incident management workflow using ServiceNow.

## 🔧 Features Implemented
- Client Script for real-time priority calculation
- Business Rule for backend data validation
- Assignment Rule for automatic ticket routing
- SLA configuration for time-bound resolution

## 🧠 Key Learnings
- Difference between Client Scripts and Business Rules
- Incident lifecycle in ITSM
- Automation of workflows in enterprise systems

## 🛠 Tools Used
- ServiceNow ITSM Module
